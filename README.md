# 村民模组

#### 介绍
Minecraft Fabric Mod, supports Minecraft 1.18.2+ . 状态：活跃 / 开源。

#### 安装教程

1.  在releases下载本模组
2.  根据版本说明，安装相应的fabric loader和API版本。
3.  将jar文件拖入versions\你的版本名\mods
4.  启动Minecraft！
